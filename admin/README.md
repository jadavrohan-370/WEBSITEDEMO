# Foodie Admin Panel

This is the professional admin panel for the Foodie Website.

## Features
- **Dashboard**: Overview of sales, orders, and stats.
- **Product Management**: Add, edit, delete products.
- **Menu Management**: Manage Starters, Mains, Desserts.
- **Messages**: View and reply to "Contact Us" inquiries.

## Setup
1. Run `npm install` to install dependencies.
2. Run `npm run dev` to start the development server.

## Note on Backend
Currently, this Admin Panel uses `localStorage` to simulate database persistence. This allows you to test all features immediately without setting up a MongoDB backend. The data will persist in your browser.
